MAUS Parameter Set

See also the file USAGE in the top directory of the package.

This is the Russian language parameter set of MAUS (LANGUAGE=rus-RU). HMM were trained to the 
speech corpus INTAS provided by Daniil Kocharov, University of St. Petersburg (spontaneous 
speech, 7 speakers of the region around St. Petersburg).
This PARAM set contains no pronunciation model, hence only forced alignment is possible.
The input phoneme set covers the most commonly used SAM-PA sets of Russian (there is no
official Russian SAM-PA set). It also covers some additional symbols such as /T D/ to 
cover the usage of foreign words.

/pS_j/ is a clone of /tS_j/
/x_j/ is a clone of /x/
/?/ is a German acoustic model of a glottal stop
nasalized vowels a~ E~ e~ o~ are clones of their non-nasalized vowels
/T/ and /D/ are clones of /f/ and /z/
/aI/, /aU/, /EI/, /@U/ are Australian English acoustic models
